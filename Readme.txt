# DELF B2 Landing Page — French-in-Seoul

This repository hosts the static HTML and CSS for the **DELF B2 (델프 B2) preparation landing page**.  
It is part of the [French-in-Seoul](https://french-in-seoul.com/) project to support learners preparing for DELF B2 in Seoul and online.

## Features
- SEO-optimized structure and metadata for DELF B2 keywords.
- Responsive HTML5 + CSS with a simple design system.
- JSON-LD structured data (Service + FAQ) for rich results.
- Linked with [french-in-seoul.com](https://french-in-seoul.com/) for cross-site authority.

## Deployment
Upload `index.html` and `delf-b2.css` to hosting or serve via GitHub Pages.  
Domain can be mapped directly or served via the repository’s GitHub Pages URL.
